# 智慧停车行业标准技术指标

> 写标书时引用以下真实指标，不要编造。

---

## 车牌识别系统

| 指标 | 标准参数 |
|------|---------|
| 摄像机像素 | ≥400万 |
| 识别率（白天） | ≥99.5% |
| 识别率（夜间） | ≥99.0%（补光灯辅助） |
| 识别速度 | ≤200ms/辆 |
| 车牌类型支持 | 蓝牌/黄牌/新能源绿牌/港澳双牌/领事馆牌/警牌/军牌 |
| 无牌车方案 | 扫码入场+出场扫码缴费 |

## 车位引导系统

| 指标 | 标准参数 |
|------|---------|
| 探测方式 | 超声波+地磁双模 |
| 室内引导屏 | LED，分区显示空余车位数 |
| 室外诱导屏 | 三级（道路→街区→场库入口），全彩LED |
| 反向寻车 | 支持车牌/车位号/时间查询 |
| 室内导航 | 蓝牙AOA定位，精度≤3米 |

## 无感支付

| 指标 | 标准参数 |
|------|---------|
| 支付方式 | 微信/支付宝无感支付 + ETC + 扫码 |
| 扣费延迟 | ≤3秒 |
| 电子发票 | 自动开具，支持微信/支付宝卡包 |

## 安防监控

| 指标 | 标准参数 |
|------|---------|
| 覆盖范围 | 全场无死角 |
| 分辨率 | ≥1080P |
| AI分析 | 违停检测/跨线报警/人员徘徊识别 |
| 存储周期 | ≥30天 |

## 平台性能

| 指标 | 标准参数 |
|------|---------|
| 平台可用率 | ≥99.9% |
| 系统响应时间 | ≤2秒（抬杆到过车完成） |
| 并发接入 | 单服务器≥100车道 |
| 数据存储 | 本地30天 + 云端1年 |

## 停车场分级配置标准

| 规模 | 车位数 | 出入口 | 推荐配置 |
|------|--------|--------|---------|
| 小型 | < 200 | 1进1出 | 基础车牌识别+扫码支付 |
| 中型 | 200-1000 | 1进2出或2进2出 | +车位引导+安防监控+云平台 |
| 大型 | 1000-5000 | 2进2出起 | +反向寻车+诱导屏+ETC |
| 超大型 | > 5000 | 多进多出 | +室内导航+AI视频分析+定制平台 |
