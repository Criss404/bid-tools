# 知识库说明

这是 AI 标书生成的数据底座。知识库质量直接决定标书质量。

## 文件说明

| 文件 | 作用 | 谁维护 |
|------|------|--------|
| `company.yml` | 公司资质、业务范围 | 行政/资质管理 |
| `projects.yml` | 历史中标项目业绩 | 市场/项目管理 |
| `team.yml` | 核心人员资质证书 | 人力资源 |
| `solutions/*.md` | 技术方案模板（停车系统/施工等） | 技术部 |
| `terms.json` | 行业术语词典 | 技术部 |

## 维护原则

1. **真实准确**：AI 会逐字引用这些内容，虚构=违法
2. **及时更新**：资质到期、新项目中标、人员变动后立即更新
3. **版本管理**：建议用 git 记录每次修改
4. **保密**：此目录含企业商业信息，勿上传公开仓库

## 使用方式

AI 在生成标书时会自动加载这些文件作为上下文。知识越多、质量越高，AI 生成的标书越能用。

起步阶段至少填好 `company.yml` 里的核心资质信息，其余可逐步积累。
