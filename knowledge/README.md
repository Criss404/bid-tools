# 知识库说明

> [← 返回项目主页](../README.md)

**此目录为测试模板，不含真实商业数据。使用前请替换为你自己的公司信息。**

## 文件说明

| 文件 | 作用 |
|------|------|
| `company.yml` | 公司资质、业务范围 — 测试数据，请替换 |
| `projects.yml` | 历史中标项目业绩 — 测试数据，请替换 |
| `team.yml` | 核心人员资质证书 — 测试数据，请替换 |
| `solutions/*.md` | 技术方案模板（停车系统/施工等） — 通用参考 |
| `laws/` | 招投标法规要点 — 通用知识，可复用 |
| `rules/` | 废标风险/暗标规范/评分标准 — 通用知识，可复用 |
| `industry/` | 行业技术指标 — 通用参考 |
| `templates/` | 章节大纲详解 — 通用参考 |
| `imported/` | 导入的外部文件 (pdf/docx/md) |

## 维护原则

1. **真实准确**：AI 会逐字引用这些内容，编造资质 = 违法
2. **及时更新**：资质到期、新项目中标后立即更新
3. **不要上传真实数据到公开仓库** — 此目录含占位符，替换后用本地 `~/.bid_tool/knowledge/` 维护
