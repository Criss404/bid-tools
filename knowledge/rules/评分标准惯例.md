# 技术标评分标准惯例

> 各章节分值权重参考。具体以招标文件为准，此为行业惯例。

---

## 施工类项目技术标 (满分30分)

| 章节 | 分值 | 评审要点 |
|------|------|---------|
| 项目组织与管理 | 1分 | 管理机构设置、管理思路 |
| 设计管理与方案 | **22分** | 设计管理+建筑+机电+重难点 |
| 工程施工 | 5.5分 | 施工组织+人力设备+质量+文明+进度 |
| 工程采购 | 1分 | 采购管理+材料设备方案 |
| 配合方案 | 0.5分 | 监理配合+第三方检测+竣工验收 |

## 评分等级标准

```
优 (90-100%): 方案先进、措施得力、针对性强
良 (80-90%):  方案合理、措施可行
中 (60-80%):  方案基本合理、措施基本可行
差 (<60%):    方案存在缺陷、缺项较多（评委须书面说明）
无 (0分):     完全缺项
```

---

## 写标书时的优先级

```
1. 废标条款 → 100% 响应（不响应直接0分）
2. 评标打分项 → 80% 完善（核心得分区）
3. 锦上添花 → 20%（创新方案、BIM应用等）
```

**永远先保不废标，再争高分。**
