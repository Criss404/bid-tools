# 暗标格式规范

> 暗标 = 匿名评审，投标文件中不得出现任何可识别投标人身份的标记。违反直接0分。

---

## 核心禁止项

- 不得出现投标人名称、简称、英文缩写
- 不得出现投标人徽标、LOGO、标识
- 不得出现人员姓名、项目负责人姓名
- 不得出现联系电话、邮箱、地址
- 不得出现公司网址
- 不得出现可识别的业绩项目名称（除非用"某商业综合体"替代）
- 不得出现彩色文字、彩色图片、彩色图表（黑白灰除外）

## 格式要求

| 项目 | 要求 |
|------|------|
| 纸张 | A4 白底 |
| 字体 | 宋体四号（正文），黑色；标题可用三号加粗 |
| 行距 | 固定值25磅 或 1.15倍行距 |
| 页边距 | 上3cm 下3cm 左2cm 右2cm |
| 页码 | **不得有页码** |
| 页眉页脚 | **不得有页眉页脚** |
| 目录 | **不得有目录** |
| 装订 | 胶装，统一封面格式 |

## AI 生成暗标注意事项

1. 所有"我公司"→"投标人"
2. 所有"XX智慧停车"→【投标人名称】（如招标文件允许）
3. 业绩案例不能提具体项目名称，用"某商业综合体""某三甲医院"代替
4. 人员资质只写证书类型和编号格式，不写具体姓名
5. 图片只能用黑白示意图，不能用实物照片
