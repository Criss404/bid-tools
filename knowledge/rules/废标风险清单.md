# 废标风险清单 — 投标准备必查

> 85%的废标源于"招标文件响应性缺失"。生成标书时必须逐条对照检查。

---

## 一、资格资质类

1. 营业执照过期 / 法人与营业执照不一致
2. 资质证书缺增项 / 等级不够 / 过期
3. 安全生产许可证过期 / 未年审
4. 项目经理证书等级不符 / 有在建项目
5. 未提供同类项目业绩证明 / 业绩类型不匹配
6. 社保证明未盖章 / 日期错误

## 二、签章授权类

7. 投标函未手签（用印章或电子签名代替）
8. 单位盖章缺失 / 漏页
9. 授权委托书日期晚于开标日期
10. 不同文件授权人签字笔迹不一致
11. 公证书未提供原件

## 三、保证金与财务类

12. 保证金金额多汇/少汇
13. 保证金汇入账户与招标文件不符
14. 银行回单不清晰
15. 财务报表年份不全 / 流动比率不达标
16. 联合体缺牵头方财务报表

## 四、商务技术文件类

17. 商务标与技术标项目经理不一致
18. 工期超出招标要求 / 漏填工期
19. 质量目标漏填 / 低于招标要求
20. 未响应"★"号强制条款
21. 技术参数偏离强制性规定
22. 人员职称/年龄不达标

## 五、格式与报价类

23. 超限价报价
24. 唱标函与投标函报价不一致
25. 费率计取错误
26. 页码不连续 / 跳页
27. 未使用指定封面/字体/字号
28. 装订方式不符合要求（如用活页夹代替胶装）
29. 正副本内容不一致

## 六、递交类

30. 超过截止时间送达
31. 送错开标地点
32. 指定代表未到场 / 未带身份证
33. 电子版U盘不可读

---

## 自查口诀（标书提交前必过）

```
签章齐不齐 资质过不过
报价超不超 工期对不对
条件应不应 格式合不合
时间赶不赶 地点准不准
```

**生成标书时，AI 应逐条标注上述风险项的处理状态：[已规避] / [待人工确认] / [不适用]**
